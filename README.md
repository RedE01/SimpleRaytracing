# SimpleRaytracing
Made in C++ using OpenGL.

## Features
* Reflective and non reflective walls.
* Walls cast shadows on eachother.
* Reflective ground and ceiling.

## Compiling with gcc
Use `g++ -o Raytracing src/Main.cpp -lglfw -lGL -lGLEW` in OpenGLRaytracing folder.
